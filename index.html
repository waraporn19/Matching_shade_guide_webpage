Abstract


   We develop the algorithm by developing an automatic denture color selection system through a mobile camera to quickly match the color of the teeth and promote learning to choose denture colors for shade guide students using artificial. intelligence (AI), the system uses training data and denture image testing to create a model. The method starts by taking a picture of the denture (Input) and calculating the color value of each denture. It is then imported into the artificial intelligence (AI) process to categorize according to the characteristics of each denture color value. Output: There is a shade guide matching notification application, consisting of 1. Matching guide shade name 2. CIE L a b 3. Similar shade guides 4. Similar shade guides 5. Equivalent to the 3D Master shade guide. The system also collects tooth color analysis data in the database for further analysis.
